# AI Choose Your Adventure RPG

Browser-based RPG with dynamic story generation via LLM.  
Built with Next.js 15, TypeScript, Zustand, Tailwind CSS, and OpenAI API.

## Features
- Character editor (class, stats, inventory)
- Dynamic scenes generated by AI based on player choices and character state
- Streaming text responses for immersive experience
- Auto-save and multiple save slots (IndexedDB)
- Responsive dark-themed UI

## Tech Stack
- Next.js App Router
- TypeScript (strict)
- Zustand (global state)
- TanStack Query (async state)
- Vercel AI SDK + OpenAI
- Tailwind + shadcn/ui
- Framer Motion
